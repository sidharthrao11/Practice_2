Ways to Apply CSS

There are three types to apply CSS

  • Inline CSS
  • Internal CSS
  • External CSS

1. Inline CSS : its the technique to give style to a single element.
just type style tag within the HTML element and apply style into the code.

2. Internal CSS : for giving internal CSS we need to type style tag in HEAD section.
and select elements using selectors and give them style.

3. External CSS : for giving external CSS we need to creat an file with .css extension.
and type HTML code in another html document.
and link css file into the html file.

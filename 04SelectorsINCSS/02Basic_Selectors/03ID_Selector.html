NOTE :- when we want to give diffrent diffrent style to perticular element. like we have multiple paragraph
so when we apply style on the paragraph tag it'll be applied same to the all paragraph tag. but if we want to
give each paragraph tag diffrent style we use ID selector.
for using id selector we need to define # before the tag name for exp:- #p{ type style here }.
each ID should be unique.

in the below example we've two paragraph tags but when we give diffrent id to both and apply diffrent style
diffrent style applied to the both paragraph(this is the main use of ID selector.)

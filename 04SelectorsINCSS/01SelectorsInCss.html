Selectors in CSS

1. Basic Selectors
  • Universal Selectors(*)
  • ElementS selectors(Tag Name)
  • ID Selectors(#)
  • Class Selectors(.)
  • Attribute Selectors(attribute name)

2.Combinator Selectors
  • Decendant Selector(one space)
  • child Selector (>)
  • Adjacent Sibiling Selector(+)
  • General Sibiling Selector(~)

3. Group Selectors( , )
  • Group Selector

4. Pseudo Selectors
  • Pseudo Class Selector(:)
  • Pseudo Element Selector(::)
